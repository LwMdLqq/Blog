#### 什么是闭包
定义在一个函数内部的函数。本质上，闭包就是将函数内部和函数外部连接起来的一座桥梁
#### 变量的作用域
全局变量和局部变量
在js语言中，函数内部可以直接读取函数外部的全局变量，而函数外部却无法读取函数内部的变量
如何获取函数内部的变量呢？那就是在函数内部，再定义一个函数，这个函数可以拿到它上一级函数的变量，接着把这个函数当成返回值返回，就可以拿到定义在函数内部的变量

```
function fn() {
    var a = 100
    return function fun2() {
        console.log(a)
    }
}
var result = fun1()
```
